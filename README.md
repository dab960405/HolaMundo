# 📱 Hola Mundo — Android Studio

Primera app Android desarrollada como parte del Módulo 2: Kits de desarrollo de software para dispositivos móviles.

## 🛠️ Tecnologías

- Android Studio
- Kotlin
- Dispositivo de pruebas: Samsung Galaxy S21 FE (físico, conectado vía ADB)

## ✨ Funcionalidad

App interactiva que va más allá del "Hello World" tradicional:

- Botón que cambia el texto del saludo en cada clic (rotando entre varias frases)
- Contador de clics en pantalla
- Cambio dinámico del color de fondo
- Notificación tipo Toast al presionar el botón

## 📂 Estructura del proyecto

- `app/src/main/java/.../MainActivity.kt` — lógica principal de la app
- `app/src/main/res/layout/activity_main.xml` — diseño de la interfaz
- `app/src/main/AndroidManifest.xml` — configuración y permisos de la app

## 🚀 Cómo ejecutar

1. Clonar el repositorio:
```bash
   git clone https://github.com/tu-usuario/HolaMundo.git
```
2. Abrir el proyecto en Android Studio:
   - `File → Open` y seleccionar la carpeta clonada
3. Esperar a que Gradle sincronice las dependencias (puede tardar unos minutos la primera vez)
4. Conectar un dispositivo físico vía USB con la depuración USB activada, o seleccionar un emulador (AVD)
5. Ejecutar la app:
   - `Run → Run 'app'` o presionar el botón ▶️
